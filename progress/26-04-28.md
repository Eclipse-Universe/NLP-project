## output20 제출 결과 분석

- output20 (Qwen v5 단독): **52.5391점** (v3 53.30 대비 -0.76점)
- output21 (output16+output20 앙상블): **53.4659점** ← 신규 최고 (+0.08점)

### 앙상블 이득이 미미한 원인 (연구 기반 분석)
1. **RougeEvalCallback greedy/beam 불일치**: 콜백은 greedy 평가, 추론은 beam=4+lp=0.7 → 잘못된 checkpoint 선택
2. **길이 기반 앙상블 한계**: content-agnostic, 길이만 맞아도 내용 틀리면 ROUGE 개선 없음
3. **data leakage**: v5 holdout이 학습 데이터에 포함 → epoch3 ROUGE=1.49 신뢰 불가

### 연구 자료 기반 다음 방향 (우선순위 순)
1. **RougeEvalCallback beam 수정 (EASY, HIGH ROI)**: 추론 조건과 동일하게 beam=4+lp=0.7로 checkpoint 선택 → v6 재학습
   - 논문: "Watch Out For Your Beam Search Hyperparameters" (kaitchup 2024)
   - 예상: +0.5~2.0점
2. **min_new_tokens=25 (EASY, INSTANT)**: 너무 짧은 요약 방지 → v3로 즉시 재추론 (output22)
   - 논문: "Length Controlled Generation" (arxiv 2412.14656)
3. **MBR Decoding (MEDIUM)**: N개 샘플 생성 후 ROUGE 합의 최고 선택
   - 논문: "Follow the Wisdom of the Crowd" (ACL Findings 2023)
   - 예상: +1~3점
4. **AdaLoRA (EASY, 재학습)**: 레이어별 rank 적응적 할당
   - 논문: AdaLoRA (ICLR 2023), 예상: +0.3~1.0점
